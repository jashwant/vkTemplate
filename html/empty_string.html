Empty String

Parameter "state" in JSON is empty, but it doesn't crash application.

JSON:
{ "first_name":"Vadim", "last_name":"Kiryukhin", "state":""}
Template:
My name is <%=first_name%> <%=last_name%>. Greeting from <%=state%>!
Function:
$('#leftpanel').vkTemplate('basic.tmpl','empty_string.php');
