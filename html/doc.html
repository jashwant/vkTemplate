Documentation

  • No configuration is needed
  • Include in the page head: <script type="text/javascript" src="vktemplate.js"></script>
  • Usage format: $('#container').vkTemplate(urlTemplate, jsonData [, callback]);
  • Parameters:
    • @urlTemplate - URL address to get template;
    • @jsonData:
      - URL address to get data.
      - JSON string;
    • @callback - callback function (optional)
  • Usage Examples:
    • $('#container').vkTemplate(urlTemplate, urlJsonData);
    • $('#container').vkTemplate(urlTemplate, '{"foo":"bar"}');
    • $('#container').vkTemplate(urlTemplate, urlJsonData, function(){...});
  • Template syntax:
    • javascript logic is wrapped with <% ... >
    • stuff which should be printed is wrapped with <%= ... >
  • Template Examples:
    • <%=first_name%> <%=last_name%>
    • <%for(var ix=0;ix<time.length;ix++) {%>
  time is: <%=time[ix].h%>:<%=time[ix].m%>:<%=time[ix].s%>
<%}%>
