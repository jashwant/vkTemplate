Callback

After template is rendered, optional callback function can be called. With this function you can resize existing and/or loaded elements, adjust layout, hide loading image, etc.

JSON:
{ "first_name":"Vadim", "last_name":"Kiryukhin", "state":"California"}
Template:
My name is <%=first_name%> <%=last_name%>. Greeting from <%=state%>!
Function:
$('#leftpanel').vkTemplate('basic.tmpl','basic.php', function(elm,jsonObj){$(elm).children().css('background-color','yellow')});
