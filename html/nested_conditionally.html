Conditionally Nested

Template basic.tmpl is loaded only if seconds is odd number. Click on "Conditionally Nested" menu item several times to see how it works.

JSON:

{"h":"06","m":"10","s":"05"}
Template:

 Current time is: <%=h%>:<%=m%>:<%=s%>

<%if (s%2) {%> 
  odd second
  <script type="text/javascript">
    $('#subcontainer').vkTemplate('basic.tmpl','basic.php');
  </script>
<%} else {%> 
  even second
<%}%>
Function:
$('#leftpanel').vkTemplate('nested_conditionally.tmpl','condition.php');
