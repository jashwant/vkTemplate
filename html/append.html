Append Technique

All examples above demonstrate "replace" technique - all content of the element-container ( the element which plugin is attached to) is replaced with template stuff.

But very often we need to keep container's content and append or prepend template to the content rather than replace it. This example shows how to do this.

JSON:
{ "first_name":"Vadim", "last_name":"Kiryukhin", "state":"California"}
Template:
My name is <%=o.first_name%> <%=o.last_name%>. Greeting from <%=o.state%>!
Function:
$('#leftpanel')
	.append('<div>')
	.children(':last')
	.vkTemplate('tmpl/basic.tmpl','php/basic.php');
