Append Technique

All examples above demonstrate "replace" technique - all content of the element-container ( the element which plugin is attached to) is replaced with template stuff.

But very often we need to keep container's content and append or prepend template to the content rather than replace it. Say, we want to add another item to a list. This example shows how to do this.

In this function <span> element is used as a parent element to wrap template content. You can keep it or get rid of it as it's shown in this example. Not only <span>, but any element can be used as a wrapper.

JSON:
{ "first_name":"Vadim", "last_name":"Kiryukhin", "state":"California"}
Template:
My name is <%=o.first_name%> <%=o.last_name%>. Greeting from <%=o.state%>!
Function:
$('<span>').vkTemplate('tmpl/basic.tmpl','php/basic.php', function(elm){
		$('#leftpanel').append($(elm).html());
	});
