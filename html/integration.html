Integration

This is an example of a very simple web application, which performs periodical updating functionality. Every 2 seconds it takes time from a server and display it. I use jQuery Smartupdater plugin to implement polling mechanism.

JSON:

{"h":"06","m":"10","s":"05"}
Template:

<%=o.h%>:<%=o.m%>:<%=o.s%>
Function:

<div id="timedisplay"></div>

<script type="text/javascript">
  $("#timedisplay").smartupdater({
      url: 'php/get_time.php', // url to data 
      minTimeout: 2000         // 2 seconds
    }, function (data) {
      $('#timedisplay').vkTemplate('tmpl/time.tmpl',data);
    }
  );
</script>
