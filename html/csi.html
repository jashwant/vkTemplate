Client Side Include

Client Side Include ( CSI ) is similar to "Server Side Include" ( SSI ) technology. Simply include actual templates into template-container. Load this template-container either as a standalone html page or with jQuery ajax's .load() function. You don't need to use vkTemplate() plugin to load template-container itself.

JSON:

no JSON
Template:

<div id="foo">
  <script type="text/javascript">
      $('#foo').vkTemplate('basic.tmpl','basic.php');
  </script>
</div>

<div id="bar">
  <script type="text/javascript">
      $('#bar').vkTemplate('nested.tmpl','loop.php');
  </script>
</div>
Function:
$('#leftpanel').load('csi.tmpl');
