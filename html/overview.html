Overview

vkTemplate is a very small, very simple and very powerful template solution for web applications.

  • Small: 65 lines of code, 1.1 k if minified

  • Simple interface: only one function with 2 parameters: template and data (the third parameter: callback function is optional)

  • Power: well known and popular micro-template engin written by John Resig ( http://ejohn.org ) is used in this plugin as a core function.
Template syntax:
  • javascript logic is wrapped with <% ... >

  • stuff which should be printed is wrapped with <%= ... >
Data format is: JSON
